Allows new users to register /users POST A JSON object holding data about the user to add, structured like: { id: 1, name: "Jake", favoriteMovies: ["Iron Man"] } None A JSON Object holding data of a particular user that was added and structured like the example to the left
Allows new users to update username /users/:id PUT None :id A JSON Object holding data of a particular user that was updated and structured like the example to the left
Adds data to add new movies to the list /users/:id/:movieTitle POST A JSON object holding data about the user to add, structured like: { id: 1, name: "Jake", favoriteMovies: ["Iron Man"] } :id/:movieTitle a text message displaying: ${movieTitle} has been added to user ${id}'s array
Removes movies from a list by id /users/:id/:movieTitle DELETE None :id/:movieTitle a text message displaying: ${movieTitle} has been removed to user ${id}'s array
Allows users to unregister /users/:id DELETE None :id user ${id} has been deleted
